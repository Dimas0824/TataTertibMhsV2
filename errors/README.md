# Error Templates

Template fallback error HTTP yang dipakai server/web router.

## File

- `303.php`: response redirect/see-other fallback.
- `400.php`: halaman untuk Bad Request.
- `403.php`: halaman untuk Forbidden.
- `404.php`: halaman untuk Not Found.
- `500.php`: halaman untuk Internal Server Error.
- `503.php`: halaman untuk Service Unavailable.
