# Storage

Folder untuk data runtime non-public aplikasi.

## Struktur

- `keys/`: penyimpanan key/token app (mis. `app_token.key`) untuk kebutuhan kriptografi token.
- `uploads/`: penyimpanan file upload runtime non-public.
 	- `uploads/news/`: file gambar berita.

## Catatan

- Jangan commit key sensitif ke repository publik.
- Pastikan permission folder sesuai kebutuhan runtime.
