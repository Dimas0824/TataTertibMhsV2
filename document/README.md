# Uploaded Documents

Folder runtime untuk dokumen/file yang diunggah user.

## Struktur

- `news/`: menyimpan gambar berita hasil upload admin.
- file lain di root `document/`: dokumen pendukung umum yang bisa diunduh sesuai hak akses.

## Catatan

- Folder ini harus writable oleh web server.
- Konten file dapat berubah sesuai aktivitas upload pengguna.
