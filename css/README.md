# CSS Assets

Stylesheet UI aplikasi berdasarkan modul halaman/komponen.

## File

- `global.css`: style global (reset, layout dasar, utilitas umum).
- `homepage.css`: style landing page publik dan admin-home style dasar.
- `login.css`: style halaman login.
- `list-tatib.css`: style halaman list tatib publik.
- `tatib-admin.css`: style halaman tatib admin.
- `pelanggaran-page.css`: style dashboard pelanggaran (mahasiswa/dosen).
- `pelaporan.css`: style form pelaporan/edit pelaporan.
- `notifikasi.css`: style halaman notifikasi.
- `news-admin.css`: style list berita admin.
- `news-form.css`: style form tambah/edit berita.
- `news-detail.css`: style halaman detail berita publik.
- `modal.css`: style modal legacy pada modul pelanggaran.
- `app-modal.css`: style modal feedback global.
- `admin-confirm-modal.css`: style modal konfirmasi admin.
- `pelaporan-cancel-modal.css`: style modal batal pelaporan.
- `error-page.css`: style halaman error custom.
- `not-found.css`: style tambahan untuk variasi halaman tidak ditemukan.
